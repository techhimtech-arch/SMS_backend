# Student Portal Dashboard - Complete Feature Guide

## Overview
This document outlines all the features and information that should be visible to students when they log in to the School Management System. The student portal serves as a comprehensive dashboard for academic activities, personal information, and school communications.

## 🎯 Core Dashboard Sections

### 1. **Welcome Header**
- **Student Name**: Full name display
- **Profile Picture**: Student photo (if uploaded)
- **Current Date & Time**
- **Quick Actions**: Logout, Profile Settings, Notifications

### 2. **Academic Overview Card**
- **Class & Section**: Current class and section assignment
- **Roll Number**: Student's roll number in class
- **Academic Year**: Current academic year
- **Enrollment Status**: Active/Inactive status

### 3. **Quick Stats Dashboard**
- **Today's Attendance**: Present/Absent status
- **Pending Assignments**: Count of incomplete assignments
- **Upcoming Exams**: Next exam details
- **Unread Announcements**: Count of new announcements
