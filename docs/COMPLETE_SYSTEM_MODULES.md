## **مثالی استعمال کے منظرنامے:**

### **Scenario 1: Quarterly Exam**
```
1. Create Exam (FINAL_TERM)
2. Assign Subjects & Papers
3. Schedule Exam
4. Enter Marks
5. Generate Results
6. Publish Results
7. Send Notifications to Parents
```

### **Scenario 2: Weekly Quiz**
```
1. Create Quiz (QUIZ exam type)
2. Set Questions/Papers
3. Conduct Quiz
4. Enter Marks
5. View Results
6. Send Notifications
```

### **Scenario 3: Monthly Assignment**
```
1. Create Assignment
2. Publish to Students
3. Students Submit
4. Teacher Grades
5. View Grades
6. Late Submissions (if allowed)
7. Final Grades Published
```

### **Scenario 4: Fee Collection**
```
1. Create Fee Structure
2. Assign Fees to Students
3. Send Reminders
4. Record Payments
5. Generate Receipts
6. Track Overdue
7. Reports to Admin
```
