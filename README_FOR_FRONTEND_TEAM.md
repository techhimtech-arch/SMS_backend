## вЪ†пЄП Important Points

### Security
```
вЬЕ JWT Token in localStorage
вЬЕ Authorization header in ALL requests
вЬЕ Handle 401 вЖТ Redirect to login
вЬЕ Handle 403 вЖТ Show "Access Denied"
вЬЕ NEVER store password
```

### Error Handling
```

403 Forbidden вЖТ Parent not linked to this student
404 Not Found вЖТ Student not found
500 Server Error вЖТ Show "Please try again later"
Network Error вЖТ Show "Could not connect to server"
```

### Data Formatting
```
Dates: Use toLocaleDateString() or Day.js
Currency: Prefix with вВє, use commas (вВє1,50,000)
```
